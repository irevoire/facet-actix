## Facet-actix

Glue code between [facet](https://facet.rs) and [actix-web](https://actix.rs/).

## Features

- Json:
  - Lets you extract json from the payload using the [Json](Json) wrapper.
  - Lets you return json using the [Json](Json) wrapper.
